# Requirements ŌĆö Agile Task Manager

## Runtime Requirements

### .NET SDK
- **Version**: .NET 9.0 (SDK 9.0.x or later)
- **Download**: https://dotnet.microsoft.com/download/dotnet/9.0

### Database
- **SQL Server** (any of the following):
  - SQL Server 2019 / 2022 (recommended for production)
  - SQL Server Express (free, dev/test)
  - SQL Server LocalDB (included with Visual Studio ŌĆö zero config)
- **Minimum version**: SQL Server 2016+ (required for JSON support)

## External Service Configuration (Optional)

These services are **optional**. The app runs fully without them ŌĆö features degrade gracefully.

### Email (MailKit / Gmail SMTP)
Required only if you want transactional emails.
```
SmtpHost:    smtp.gmail.com
SmtpPort:    587
Username:    your-gmail@gmail.com
Password:    your-gmail-app-password   ŌåÉ NOT your login password; use an App Password
```
> Gmail App Password: Google Account ŌåÆ Security ŌåÆ 2-Step Verification ŌåÆ App Passwords

### File Storage (Cloudinary)
Required only for task attachment uploads. Without it, file upload features are disabled.
```
CloudName:   your-cloud-name
ApiKey:      your-api-key
ApiSecret:   your-api-secret
```
> Sign up free at https://cloudinary.com

### AI Assistant (OpenAI)
Required only for AI-generated task descriptions/titles/tags. Without an API key, the app
falls back to built-in rule-based suggestions automatically.
```
Provider:    OpenAI
ApiKey:      sk-...
Model:       gpt-3.5-turbo    (or gpt-4o, etc.)
Endpoint:    https://api.openai.com/v1/chat/completions
MaxTokens:   500
Temperature: 0.7
```
> OpenAI API keys: https://platform.openai.com/api-keys

## Quick Setup Steps

```bash
# 1. Restore EF CLI tool (one-time per machine)
dotnet tool restore

# 2. Restore NuGet packages
dotnet restore

# 3. Run the application (auto-applies migrations + seeds demo data)
dotnet run
```

The app will be available at: `https://localhost:5001`

> **No manual `dotnet ef database update` needed** ŌĆö migrations are applied automatically on startup.
